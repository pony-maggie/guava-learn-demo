## 什么是guava

Guava是一个开源的Java库，Google Guava源于2007年的"Google Collections Library"。这个库是为了方便编码，并减少编码错误。这个库用于提供集合，缓存，支持原语句，并发性，常见注解，字符串处理，I/O和验证的实用方法。

## 程序员为啥推荐使用guava

学习使用Google Guava可以让你快乐编程，写出优雅的JAVA代码！
